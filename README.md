# Crop-Production-Climate-Agriculture-Analysis-using-Python-SQL-ML-and-Tableau
This project focuses on analyzing crop production and climate agriculture data using Python, SQL, Machine Learning, and Tableau. The dataset was first cleaned and preprocessed in Python, then divided into Crop Production and Climate Agriculture datasets. SQL was used for analytical queries, Tableau for dashboard insights, and ML models like KNN and Random Forest were used to predict crop yield categories.
