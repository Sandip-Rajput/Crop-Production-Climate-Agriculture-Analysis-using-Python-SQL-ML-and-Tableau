# Crop Production & Climate Agriculture Analysis

## Project Overview
This project analyzes crop production and climate agriculture data to understand how factors like rainfall, fertilizer, pesticide usage, area, and production affect crop yield.

## Tools & Technologies Used
- Python
- Pandas, NumPy
- SQL
- Tableau
- Machine Learning
- KNN Classifier
- Random Forest Classifier

## Project Workflow
1. Data Cleaning using Python
2. Missing Value Treatment
3. Outlier Detection
4. Dataset Splitting into:
   - Crop_Production.csv
   - Climate_Agriculture.csv
5. SQL Analysis
6. Tableau Dashboard Creation
7. Machine Learning Model Building

## SQL Analysis
Some SQL insights:
- Total production by state
- Average annual rainfall
- Crops with production greater than 50,000
- State-wise production and fertilizer usage
- Crops grown where rainfall is above average

## Machine Learning
Target Variable: Yield_Category

Models Used:
- KNN Classifier
- Random Forest Classifier

Model Evaluation:
- Accuracy Score
- Confusion Matrix
- Classification Report

## Tableau Dashboard
Created interactive dashboard using:
- Filters
- Actions
- Crop-wise analysis
- State-wise production
- Rainfall and yield insights

## Conclusion
This project helps understand the relationship between climate factors and crop production. It also demonstrates complete data analytics workflow from Python cleaning to SQL analysis, Tableau visualization, and ML prediction.
